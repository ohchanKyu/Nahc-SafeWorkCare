### 프로젝트 소개
Safe WorkCare는 산업재해 예방을 위한 통합 정보 제공 플랫폼입니다. <br>
산업재해 통계 시각화, 실제 사고 사례 분석, 사업장 맞춤형 위험 진단, <br>
예방조치 체크리스트 작성, 그리고 고용노동부 공식 자료 기반 AI 상담 기능까지 제공하여, <br>
사업장 규모나 업종에 관계없이 누구나 쉽게 안전관리를 체계화할 수 있도록 지원합니다. <br>
